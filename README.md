# WannaGo

WannaGo est une application mobile qui vous permettera de trouver une destination pour vos voyages en vous 
proposant des photos de celle-ci, des hotêls/restaurants a proximités, ainsi que les meilleurs moyens pour 
s'y rendre.
Cela sera un moyen simple de décider de votre destination en connaissant les emplacements comme s'y vous y 
aviez déja été!


Introduction de l’application

Une application permettant de lier réseau social et voyage. Il utilisera l’API de Instagram pour récupérer des photos en corrélation avec des destinations. Exemple photo de paysages, restaurants, monuments, hôtels, etc…
L’utilisateur voyageur, pourra ainsi regarder des photos en fonction d’une destination. Il aura également la possibilité de trouver les différents moyens de locomotion pour y parvenir en fonction de différents critère, temps, prix, escale, ect… De même il pourra noter la destination, les activités, ect… 
(Faire un feedback)

L’utilisateur commerçant, pourra quant à lui inscrire son restaurant, logement, hôtel, ect… En y ajoutant des photos, une description, horaire, tarifs, etc…

Matériels

-	Serveur – Base de donnée
-	API externe : Instagram – Facebook – AirFrance KLM – (Pour vérification entreprise)
-	API interne, pour les users – gestion application
